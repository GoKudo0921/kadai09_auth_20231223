# Flutterを使ったポイント獲得アプリ

## 紹介と使い方
- ポイントを獲得するアプリです
- ポイントが書かれた画像をタップするとポイントが獲得できます
- 3回タップすると一度広告が表示されます
- ブースト機能がついてます
  - 広告を閲覧して、５ポイント獲得することができます
  - 別の広告を閲覧すると、獲得ポイントが倍になります

## 工夫した点
- 実際にありそうなアプリです
- ポイントを獲得するために広告が表示される
- ブーストで一時的にポイント倍増後、3回タップしたら倍増期間が終わるようにした
